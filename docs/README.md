# Fitlogger

FitLogger bridges the gap between analog training and digital analysis. It is optimised for users who record their workouts offline or via memory and want to rapidly bulk log their data later via CLI. It provides advanced analytics and cycle planning that physical notebooks lack, without the friction of mobile app interfaces.
